# Sabor Local
Um sistema FrontEnd de aplicativo de delivery para usuários.
Onde é possível:
- Fazer pedidos
- Editar informações pessoais (endereço, cartão de crédito, foto, etc)
- Aplicar cupons de desconto
- Utilizar um sistema de fidelização de pontos para descontos no pedido
E muito mais!!

Foram usados processos de SendGrid para verificação de email, além de Asaas para pagamentos e estornos.
